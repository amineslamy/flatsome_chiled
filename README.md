# Sahab Research Data Platform | پلتفرم مدیریت داده‌های پژوهشی سحاب

This is a specialized WordPress Child Theme built on top of the **Flatsome** theme. It is designed as a platform for reporting, categorizing, and managing research data.

این یک پوسته فرزند (Child Theme) اختصاصی وردپرس است که بر پایه قالب **Flatsome** توسعه یافته است. این پلتفرم با هدف گزارش‌دهی، دسته‌بندی و مدیریت داده‌های پژوهشی طراحی شده است.

---

## 🌐 English Description

### Project Overview
**Sahab** leverages the flexibility of Flatsome and extends it with custom functionalities to handle complex research data. It integrates deeply with ACF (Advanced Custom Fields) and Relevanssi to provide a robust search and data management experience.

### Key Features
- **Advanced Search:** Integrated with Relevanssi for precise search results and custom excerpts.
- **Data Management:** Custom logic for handling research-specific data types via ACF.
- **Jalali Calendar:** Full support for Persian dates using a custom Jalali datepicker.
- **Custom Dashboard:** A core system for managing research reports and user activities.
- **Performance Optimized:** Modular code structure with localized assets (fonts, CSS, JS).

### Directory Structure
- `/inc`: Core logic, including ACF automation, comment handling, and admin labels.
- `/assets`: Public and admin-side CSS/JS and typography files.
- `/templates`: Custom page templates for data display.
- `/languages`: Translation files for theme localization.

## 🇮🇷 توضیحات فارسی

### معرفی پروژه
**سحاب** یک پلتفرم تخصصی مبتنی بر وردپرس است که از انعطاف‌پذیری قالب فلتسام بهره برده و آن را با قابلیت‌های سفارشی برای مدیریت داده‌های پژوهشی گسترش داده است. این پروژه با استفاده از ACF و Relevanssi، تجربه‌ای قدرتمند در جستجو و مدیریت داده‌ها ارائه می‌دهد.

### قابلیت‌های کلیدی
- **جستجوی پیشرفته:** یکپارچه‌سازی با Relevanssi برای نمایش دقیق نتایج و چکیده‌های سفارشی.
- **مدیریت داده‌ها:** منطق اختصاصی برای مدیریت انواع داده‌های پژوهشی از طریق ACF.
- **تقویم شمسی:** پشتیبانی کامل از تاریخ هجری شمسی با انتخابگر تاریخ بومی.
- **داشبورد اختصاصی:** سیستم مرکزی برای مدیریت گزارش‌های پژوهشی و فعالیت کاربران.
- **بهینه‌سازی شده:** ساختار کدنویسی ماژولار و مدیریت بهینه منابع (فونت‌ها، استایل‌ها و اسکریپت‌ها).

### ساختار پوشه‌ها
- `inc/`: منطق اصلی برنامه، از جمله خودکارسازی ACF، مدیریت دیدگاه‌ها و برچسب‌های ادمین.
- `assets/`: فایل‌های CSS و JS سمت کاربر و مدیریت به همراه فونت‌ها.
- `templates/`: قالب‌های اختصاصی صفحات برای نمایش داده‌ها.
- `languages/`: فایل‌های ترجمه برای بومی‌سازی پوسته.
